# Strudel Sounds > Synths

Pasted from the Strudel web app's Sounds > Synths tab.

Usage: `.s("name")` selects the synth/waveform, `.n(i)` picks a specific
variation for the `gm_*` (General MIDI) entries (0-indexed, count shown is how
many variations exist). Entries with no count are single continuous waveforms
or generators (noise types, basic oscillator shapes), not sample sets.
