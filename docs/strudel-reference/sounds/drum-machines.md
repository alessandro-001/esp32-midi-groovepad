# Strudel Sounds > Drum Machines

Pasted from the Strudel web app's Sounds > Drum Machines tab.

Usage: `.s("name")` selects the sample bank, `.n(i)` picks a specific variation
(0-indexed, count shown is how many variations exist). Naming convention:
`<machine>_<part>`. Part codes: bd=bass drum, sd=snare, hh=hihat, oh=open hat,
cp=clap, cr=crash, rd=ride, rim=rimshot, cb=cowbell, ht/mt/lt=high/mid/low tom,
tb=tambourine, sh=shaker, perc=percussion, misc/fx=miscellaneous.
