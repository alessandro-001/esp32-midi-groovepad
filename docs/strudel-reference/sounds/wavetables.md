# Strudel Sounds > Wavetables

Pasted from the Strudel web app's Sounds > Wavetables tab.

Usage: `.s("name")` selects the wavetable, `.n(i)` picks a specific variation
(0-indexed, count shown is how many variations exist).

| Wavetable | Variations |
|---|---|
| `wt_digital(5)` | 5 |
| `wt_digital_bad_day(1)` | 1 |
| `wt_digital_basique(1)` | 1 |
| `wt_digital_crickets(1)` | 1 |
| `wt_digital_curses(1)` | 1 |
| `wt_digital_echoes(1)` | 1 |
| `wt_vgame(11)` | 11 |
